# Task Manager

A Node.js task management CLI for Gemini CLI training exercises.

## Setup

```bash
npm install
npm start
```

## Features

- Create, update, delete tasks
- Filter by status
- Sort by due date
- Search tasks
- JSON file persistence

## Exercise Goals

Use Gemini CLI to:
1. Explore the code architecture
2. Add a CLI interface using readline
3. Add colored console output
4. Create comprehensive Jest tests
5. Add priority levels to tasks
6. Implement task categories/tags
